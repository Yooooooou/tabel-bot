# 📋 Инструкция по настройке Табель-бота v2

---

## Файлы проекта

| Файл | Назначение |
|---|---|
| `bot.py` | Главный файл бота — весь Telegram-интерфейс |
| `sheets.py` | Работа с Google Sheets (создание, запись, экспорт) |
| `database.py` | Хранение сотрудников в файле `db.json` |
| `schedule.py` | Расчёт плановых смен по графику |
| `config.py` | Настройки и константы |
| `requirements.txt` | Зависимости Python |
| `railway.toml` | Конфиг для Railway |

---

## Шаг 1 — Telegram бот

1. Найдите **@BotFather** в Telegram
2. `/newbot` → введите имя → введите username
3. Скопируйте **токен**: `7123456789:AAH...`
4. Узнайте свой **Telegram ID** через **@userinfobot** → напишите `/start`

---

## Шаг 2 — Google Sheets

1. Создайте таблицу на https://sheets.google.com
2. Назовите: `Табель 2026`
3. Скопируйте **ID таблицы** из адресной строки:
   ```
   https://docs.google.com/spreadsheets/d/[ВОТ_ЭТО_ID]/edit
   ```

---

## Шаг 3 — Google Service Account

### 3.1 Google Cloud Console
1. https://console.cloud.google.com → создайте проект `TabelBot`
2. **APIs & Services → Library** → включите:
   - `Google Sheets API`
   - `Google Drive API`

### 3.2 Service Account
1. **APIs & Services → Credentials → + CREATE CREDENTIALS → Service account**
2. Имя: `tabel-bot` → Create → Done
3. Нажмите на созданный аккаунт → вкладка **KEYS**
4. **ADD KEY → Create new key → JSON** → скачается файл

### 3.3 Доступ к таблице
1. Откройте скачанный JSON, найдите `"client_email"`
2. Откройте Google таблицу → **Поделиться**
3. Вставьте email → роль **Редактор** → Отправить

---

## Шаг 4 — Деплой на Railway

### 4.1 GitHub
1. https://github.com → **New repository** → `tabel-bot`
2. Загрузите все файлы проекта (**Add file → Upload files**)

### 4.2 Railway
1. https://railway.app → войти через GitHub
2. **New Project → Deploy from GitHub repo** → выберите `tabel-bot`

### 4.3 Переменные окружения
**Variables** → добавьте:

| Переменная | Значение |
|---|---|
| `BOT_TOKEN` | токен из Шага 1 |
| `ADMIN_CHAT_ID` | ваш Telegram ID |
| `SPREADSHEET_ID` | ID таблицы из Шага 2 |
| `GOOGLE_CREDS_JSON` | **весь текст** JSON-файла из Шага 3 |

Для `GOOGLE_CREDS_JSON` — откройте скачанный JSON файл в текстовом редакторе, выделите **весь текст** (Ctrl+A), скопируйте и вставьте как значение переменной.

---

## Шаг 5 — Первый запуск

1. Напишите боту `/start`
2. Перейдите в **👥 Сотрудники → ➕ Добавить сотрудника** и добавьте всех
3. Перейдите в **📊 Таблица → 🔨 Создать/Пересобрать** — создастся лист текущего месяца

## Как использовать бота

### Главное меню
```
✅ Отметить смены     → выбрать дату → отметить каждого сотрудника
👥 Сотрудники         → добавить / список / уволить
💰 Аванс/Удержание    → выбрать сотрудника → ввести сумму
📊 Таблица            → создать / скачать xlsx / открыть в браузере
⚙️ Настройки          → добавить/удалить администраторов бота
```

### Отметка смен
- Кнопки показывают **иконку текущего статуса**: ✅ = 1, 🔸 = частично, ⬜ = 0
- Нажмите на сотрудника → выберите значение (1 / 0.7 / 0.5 / 0.3 / 0)
- Для **раннеров** — вводите количество часов текстом (например: `8` или `6.5`)
- Для **замены** — нажмите "🔄 Это замена" → выберите кого заменяет → строка добавится автоматически

### Добавление сотрудника
При добавлении бот спрашивает:
1. ФИО
2. Телефон
3. Должность
4. Раздел (Админы/Кассиры, Официанты утро/вечер, Раннеры, Тех.отдел)
5. График (2/2 / 5/2 / 7/0 / свободный)
6. Для **5/2** — выходные дни (Вс+Пн, Сб+Вс и т.д.)
7. Для **2/2** — стартовая рабочая дата (ДД.ММ.ГГГГ)

После добавления нажмите **📊 Таблица → Пересобрать** чтобы строка появилась.

### Напоминания
Бот автоматически присылает сообщение в **08:00** и **21:00** каждый день с кнопкой перехода к отметке смен.

### Новый месяц
1-го числа каждого месяца бот **автоматически создаёт новый лист** в Google таблице. Также можно создать вручную через **📊 Таблица → Пересобрать**.

## Расчёт плановых смен

| График | Формула |
|---|---|
| **2/2** | Автоматически по стартовой дате (цикл 2 рабочих / 2 выходных) |
| **5/2** | Дней в месяце − количество выходных дней в этом месяце |
| **7/0** | = Количество дней в месяце |
| **Свободный** | Не считается (раннеры) |

## Часто задаваемые вопросы

**Как редактировать уже отмеченную смену?**
Просто отметьте снова — значение перезапишется.

**Как удалить отметку смены?**
Поставьте значение 0.

**Таблица не обновляется после добавления сотрудника?**
Нажмите **📊 Таблица → Пересобрать** — это полностью пересоздаёт структуру с актуальным списком.

**Как добавить ещё одного администратора бота?**
⚙️ Настройки → ➕ Добавить администратора → введите его Telegram ID.

**Ошибка подключения к Google Sheets?**
- Убедитесь что `GOOGLE_CREDS_JSON` содержит весь JSON текст
- Проверьте что service account email добавлен в доступ к таблице
- Убедитесь что Google Sheets API включён в Google Cloud
